# FitCale Privacy

This repository hosts the public privacy policy page for FitCale.

## Publish with GitHub Pages

1. Create a public repository named `fitcale-privacy`.
2. Upload all files from this folder to the repository root.
3. In GitHub: Settings -> Pages.
4. Under Build and deployment, choose `Deploy from a branch`.
5. Select branch `main` and folder `/(root)`.
6. Save and wait a few minutes.

Your privacy policy URL will be:
`https://trsnacar.github.io/fitcale-privacy/privacy-policy.html`
